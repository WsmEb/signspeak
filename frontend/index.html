SignSpeak — مشروع PFE

نظام ترجمة لغة الإشارة بالذكاء الاصطناعي

🎯 المشكلة

70 مليون شخص أصم وبكم في العالم يعانون من عزلة رقمية كاملة. لا توجد أداة مجانية وسهلة تعمل في المتصفح مباشرة بدون تثبيت.

⚙️ طريقة العمل

MediaPipe يكشف 21 نقطة على اليد في الوقت الفعلي. هذه النقاط تُرسل لـ FastAPI الذي يُحلّلها ويُعيد اسم الإشارة ودرجة الثقة فوراً.

🧠 التقنيات

HTML + JS للواجهة، MediaPipe Hands للكشف، Python + FastAPI للـ API، NumPy لمعالجة البيانات. كل شيء يعمل في المتصفح بدون GPU.

📈 التطوير المستقبلي

تدريب نموذج ML عميق، دعم لغة الإشارة العربية، التعرف على الكلمات الكاملة، تحويل النص إلى كلام.

بُني بـ HTML + JS + MediaPipe + FastAPI
